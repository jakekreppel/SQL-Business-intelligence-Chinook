# Chinook Music Store: Business Intelligence Analysis

## Project Objective
This project uses SQL to extract actionable insights from the Chinook database, a mock digital music store. The goal was to answer key business questions regarding customer spending, market popularity, and employee performance.

## Key Insights
* **Top Customer:** The analysis identified our most valuable customer, enabling targeted loyalty programs.
* **USA Market Trends:** Rock is the dominant genre in the USA, followed by Latin and Metal.
* **Employee Performance:** We identified the top-performing Sales Support Agent by total revenue generated.

## Technical Skills Demonstrated
* **Multi-table Joins:** Connecting Customer, Invoice, and Employee tables.
* **Aggregations:** Using `SUM()`, `COUNT()`, and `GROUP BY` to summarize data.
* **Table Aliasing:** Writing clean, readable code with SQL aliases.
